# wifi-hub

Integration hub for the Atheros AR9170 (carl9170) USB WiFi driver ecosystem.

Part of a 4-project ecosystem:

- **carl9170-driver** — Linux kernel driver patches
- **carl9170fw-custom** — Firmware (SH-2 cross-compiled)
- **linux-wifi-analyzer** — WiFi analyzer TUI (fork of wavemon)
- **wifi-hub** (this repo) — Debug tooling and integration

See [CLAUDE.md](CLAUDE.md) for architecture, build commands, and cross-repo workflow.
